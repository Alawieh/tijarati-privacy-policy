Privacy Policy / سياسة الخصوصية

1. Overview / نظرة عامة

Tijarati Pro respects your privacy. This app uses the camera only to scan barcodes for app functionality. No personal images or sensitive data are collected or shared.
تطبيق Tijarati Pro يحترم خصوصيتك. يستخدم التطبيق الكاميرا فقط لمسح الرموز الشريطية من أجل وظيفة التطبيق. لا يتم جمع أي صور شخصية أو بيانات حساسة أو مشاركتها.

2. Data Collection and Security / جمع البيانات والأمان

This app does not collect personal user data. However, anonymous usage data may be collected by Firebase Analytics to improve the app experience.
هذا التطبيق لا يجمع بيانات شخصية للمستخدم. ومع ذلك، قد يتم جمع بيانات الاستخدام بشكل مجهول بواسطة Firebase Analytics لتحسين تجربة التطبيق.

3. Data Types / أنواع البيانات

  • Camera input for barcode scanning (local only) / إدخال الكاميرا لمسح الرموز الشريطية (محلي فقط)
  • Anonymous usage data via Firebase Analytics / بيانات استخدام مجهولة عبر Firebase Analytics

4. Data Usage and Handling / استخدام البيانات ومعالجتها

The app does not share any collected data with third parties. Data collected by Firebase Analytics is used only to improve app functionality.
لا يشارك التطبيق أي بيانات يتم جمعها مع أطراف ثالثة. تُستخدم البيانات التي يجمعها Firebase Analytics فقط لتحسين وظيفة التطبيق.

لأي استفسارات بخصوص الخصوصية، تواصل معنا على: tijaratipro@gmail.com
